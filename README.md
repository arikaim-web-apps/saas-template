## SaaS template

#### Requirements 
  * [Arikaim CMS](https://github.com/arikaim/arikaim)


#### Created With 
    * Tailwind CSS
    * FlyonUi

#### Customization 

In theme folder: 

1. Install Tailwind run 

```sh

npm install

```

2. Start watch process

```sh

npx @tailwindcss/cli -i css/include.css -o css/tailwind.min.css -m --watch

```
